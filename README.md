## 🛠️ Tecnologías utilizadas
* **Lenguaje:** PHP 8.x
* **Base de datos:** MySQL / MariaDB
* **Frontend:** HTML, CSS, JavaScript (Bootstrap/Tailwind)
* **Gestión de dependencias:** Composer (si aplica)

## 📋 Requisitos previos
Para ejecutar este proyecto localmente, asegúrate de tener:
* Un servidor local como **XAMPP**, **Laragon** o **WAMP**.
* PHP instalado (versión mínima 7.4 o superior).

## 🚀 Instalación y configuración
1. Clona el repositorio:
   `git clone https://github.com/soporte2-dotcom/ingreso-inventario.git
2. Mueve la carpeta a tu directorio `htdocs` o `www`.
3. Importa la base de datos (archivo `database.sql` si lo tienes).
4. Configura tus credenciales en el archivo de configuración (el que ignoramos en el `.gitignore`).
